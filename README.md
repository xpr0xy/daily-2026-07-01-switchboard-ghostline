# switchboard ghostline

creative brief: department is phreak / RF / telecom ghosts. this is a safe synthetic DTMF routing bench shaped like an enamel-and-bakelite telephone switchboard: exchange jacks, cloth cords, tone pads, ghost lamps, crosstalk, and route logs. stack is Vite + vanilla TypeScript + SVG + Web Audio because the substrate is an instrument panel, not another dark canvas field. it is safe/public because it generates only local educational DTMF tones and synthetic routes, never touches carriers, networks, credentials, or packets. it should not look like the last builds: no optical bench, no cartographic label table, no terrain/field sim, no right rail, no dark-panel simulator shell.

## systems
- synthetic DTMF tone engine with local Web Audio playback and route logs
- SVG switchboard simulator with exchange lamps, sagging cords, crosstalk, and animated tone pulses
- concept-native controls: tone pad, reseatable cord plugs, gain/bleed/operator bias levers with readouts
